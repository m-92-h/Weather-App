# 🌤️ Weather Now

A real-time weather forecast app — search any city and get current conditions, hourly & 7-day forecasts.

![Preview](/public/preview.jpg)

---

## Tech Stack

Next.js 16 · TypeScript · Tailwind CSS v4 · Headless UI · AOS

## Features

- 🔍 City search with live suggestions
- 🌡️ Current weather + feels like, humidity, wind, precipitation
- 🕐 24-hour hourly forecast
- 📅 7-day daily forecast
- ⚙️ Unit switcher (°C/°F · km/h/mph · mm/in)
- 🎨 Light / Dark theme
- 📱 Fully responsive

## APIs

| API                    | Cache  |
| ---------------------- | ------ |
| Open-Meteo Forecast    | 30 min |
| Open-Meteo Geocoding   | 24 hrs |
| Open-Meteo Suggestions | 1 hr   |

## Getting Started

```bash
npm install
npm run dev
```
